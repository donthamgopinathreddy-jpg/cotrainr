# 3D Body Models for BMI Visualization

This folder contains GLB models used by the BMI Details screen for 3D body visualization.

## Files

- `male.glb` – Male body model (neutral stance)
- `female.glb` – Female body model (neutral stance)

## Replacing Models

To use custom models, replace `male.glb` and `female.glb` with your own GLB files. Models should be:

- Neutral stance, minimal texture size
- glTF 2.0 / GLB format
- Validated with [glTF Validator](https://github.khronos.org/glTF-Validator/)

## Default

The default models are placeholders from modelviewer.dev. Replace with proper male/female body models for production use.
